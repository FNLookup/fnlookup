About FNLookup

Learn about FNLookup.



FNLookup is a page made by tposejank on GitHub and provides a basic Fortnite browsing service.
What it can provide is listed here:
  • Item search bar
  • Live item shop
    • Daily section
    • Featured section
    • Other sections
  • Season progress bar
    • Live countdown until season ending day.
  • In-game views (from a new-player-view)

The page is free to use, and has no ads.
It's hosted on GitHub Pages, a free page hosting service provided by GitHub.

Uses as a UI base, fnbr.co.
